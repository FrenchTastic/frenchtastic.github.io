EvalObs

Application iOS de suivi de l'observance

Cette application permet d'effectuer un suivi de l'observance chez le patient, via une échelle analogique et un questionnaire anonyme.

Leuconnect

Site web ASP.NET MVC 5

Plateforme à destinations des personnes atteintes de leucodystrophie. L'application permet de communiquer entre les membres, de connaître les actualités liés à la plateforme et de s'inscrire et participer à des études cliniques.

VigiMobile

Application mobile Android avec NFC

Cette application à pour but d'aider les auxialiaires de vie à dématérialiser leur carnet de liaison, l'application se lance et détecte le patient via un tag NFC.

Clinica

Site Web ASP.NET MVC 5

Plateforme permettant la création, l'administration et le suivi d'études cliniques. L'application comporte un outil de création de questionnaires, multilingues, et couvre de nombreux rôles nécessaire dans le suivi d'une étude clinique
(promoteur, TEC, ARC, statisticien).

CardiMobile

Site web mobile en jQuery Mobile

Site web mobile réalisé en jQuery Mobile avec un backend en ASP.NET Web Api permettant d'effectuer le suivi des insuffisants cardiaques à domicile. (Envoi d'informations relatives au poids, tension, etc....)

OTP API
